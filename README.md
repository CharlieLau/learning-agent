# 从零开始构建智能体：荒岛求生指南

> 一个用故事化方式学习 AI Agent 开发的完整教程

## 📖 项目简介

这是一个独特的 AI Agent 开发教程项目，通过"荒岛求生"的隐喻故事，将复杂的技术概念转化为生动易懂的学习内容。跟随主角和 AI 助手 Friday 的冒险，从零开始掌握构建智能体的核心技能。

## 🎯 学习目标

通过本教程，你将学会：

- **理解 LLM 与 Agent 的本质区别**
- **掌握 Prompt Engineering 核心技巧**
- **构建 ReAct 推理循环系统**
- **使用低代码平台（Dify/Coze）构建工作流**
- **实现 RAG（检索增强生成）系统**
- **设计多智能体协作架构**
- **掌握 Agent 评估与对齐方法**
- **构建完整的 Agent 应用系统**
