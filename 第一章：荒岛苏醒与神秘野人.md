# 第一章：荒岛苏醒与神秘野人

—— 从 LLM 到 Agent 的觉醒

## 📖 荒岛日记：第 1 天

海浪的轰鸣声像是一台过载的服务器风扇，在我耳边疯狂旋转。我睁开眼，刺眼的阳光让我意识到——我的船沉了，我的代码库没了，这里是没有任何 API 文档的荒岛。

但我并不孤单。

在沙滩的另一头，站着一个身材魁梧的"野人"。他肌肉发达（象征着强大的算力），眼神深邃（存储着海量的预训练数据）。我试探性地对他喊了一声："你好，我是谁？"

他立刻转过头，用字正腔圆的播音腔回答："你好！你是这片沙滩上的幸存者。根据目前的环境分析，你现在的状态属于'遇难'。"

我心中狂喜！这不仅是一个人，更是一个拥有人类顶级知识库的超级助手！我给他取名叫 Friday。我的肚子开始咕咕叫，既然 Friday 这么聪明，解决温饱应该不成问题。

我指着不远处波光粼粼的海面，下达了第一个指令：

"Friday，去给我抓条鱼回来。"

Friday 点了点头，眼神充满了智慧的光芒。他看着大海，开口说道：

"抓鱼是一种通过工具或徒手从水中获取鱼类资源的行为。常见的抓鱼方法包括垂钓、网捕和叉鱼。鱼类富含蛋白质，对你目前的生存至关重要……"

五分钟过去了。

十分钟过去了。

Friday 依然站在原地，口若悬河地向我科普《世界捕鱼发展史》，但他哪怕一毫米都没有移动，甚至连脚都没有迈进水里。

"动啊！去抓啊！"我绝望地吼道。

Friday 困惑地看着我："我已完成了关于'抓鱼'的文本生成。请问还需要更多关于鱼类烹饪的信息吗？"

那一刻，饥饿感让我大脑瞬间清醒。我终于意识到，眼前的 Friday 根本不是什么"智能体"。

他只是一个大语言模型（LLM）——一个被困在躯壳里、只会预测下一个字的"缸中之脑"。

---

## 🛠️ 技术生存手册：为什么 Friday 只是个"聊天机器人"？

在这个濒临饿死的时刻，我在沙滩上捡起一根树枝，颤抖着在沙地上画出了我遭遇的"技术海难"分析图。

Friday 的问题在于，他只有大脑（LLM），却不知道如何控制手脚（Tools），也不懂得先找树枝再叉鱼（Planning）。

在《从零开始构建智能体》的定义中，我必须完成从 LLM 到 Agent 的认知跃迁。

**公式重写：**

- **现状 (Chatbot)**: `Output = LLM(Input)`
  - Friday 的表现：我输入"抓鱼"，他输出"抓鱼的定义"。他不仅没有行动能力，甚至没有"行动"这个概念。

- **目标 (Agent)**: 如果我想活下去，我就不能只把 Friday 当作一个聊天对象。我必须通过代码构建一个系统，赋予他以下能力：
  1. 感知（Perception）：让他能看见海里有鱼，而不是只能处理文本。
  2. 大脑（Brain）：利用 LLM 进行逻辑分析，而不是单纯的文本续写。
  3. 工具（Action）：让他能调用"手"去拿叉子，而不是只在嘴上说"拿叉子"。

---

## 💻 开发者日志：迈出第一步

既然这是一场技术求生，我就得用代码来重塑 Friday。在这一章，我首先要定义出 Friday 的基础类，让他哪怕在概念上不再只是一个 `print()` 函数。

我打开了那个还没被海水泡坏的笔记本电脑（虽然电量告急），写下了构建智能体的第一行伪代码：

```python
# 传统的 LLM 调用（Friday 现在的状态）
def simple_chat(user_instruction):
    # 只能返回文字，无法在这个物理世界产生任何改变
    return llm.generate_text(user_instruction)

# 我要构建的 Agent 雏形
class Agent:
    def __init__(self, name="Friday"):
        self.llm = LLM()          # 大脑
        self.memory = []          # 记忆（目前还是空的）
        self.tools = []           # 工具（目前还没制造）

    def act(self, task):
        # 1. 思考：分解任务
        plan = self.llm.think(task)

        # 2. 行动：如果需要物理交互，就调用工具
        if "捕鱼" in plan:
            # 这里的 Tool 对应现实世界的动作
            # 只有加上这一步，Friday 才会真的走进水里
            return self.use_tool("fishing_spear")
        else:
            return self.llm.speak(plan)
```

看着屏幕上的代码，我叹了口气。现在的 Friday 还是个"空架子"。

太阳快落山了，Friday 依然站在海边发呆，嘴里念叨着："根据热力学定律，现在海水正在变凉……"

我不得不自己动手抓了只螃蟹充饥。咬着生硬的蟹腿，我看着 Friday 充满潜力的背影，暗暗发誓：

明天，我要带你去那个画满壁画的古老洞穴。我要教会你，什么叫作真正的"思考"。
